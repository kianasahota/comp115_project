# comp115_project1
Python Artistic Drawing Landscape Scene 

Description:
This project creates a landscape scene all using Python turtle graphics.
It includes mountains, trees, clouds, grass, and a sun.

Features:
Functions for drawing shapes (rectangle, triangles, circle)
Functions that call shape functions to make more complex shapes and items (trees, clouds, mountains)
Birds in sky using random dots
Layered drawing (mountains behind trees) 

Screenshot:
I have attached a screenshot in this repository of the drawing as well. 

How to Run:
Make sure Python is installed
Run the file:
artistic_drawing.py

Concepts Used:
Functions
Loops
Random module
Coordinates (x, y)
Turtle graphics
